# Installation

- Clone Repo
```js
git clone https://github.com/diwasatreya/Discord-Server-Dashboard.git
```

- After cloning, type (Shell/Terminal)
```js
npm install
```

- Create `.env` file (Environment Secret in replit)
```js
TOKEN=XXXXXXXXXX
DB=xxxxxxxxxx
```

- Fill `config.json`
